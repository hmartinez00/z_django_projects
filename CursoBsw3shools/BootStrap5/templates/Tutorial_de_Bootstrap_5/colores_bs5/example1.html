Colores Contextuales

Utiliza las clases contextuales para proporcionar "significado a través de los colores":

text-muted (Este texto está enmudecido).

text-primary (Este texto es importante).

text-success (Este texto indica éxito).

text-info (Este texto representa alguna información).

text-warning (Este texto representa una advertencia).

text-danger (Este texto representa peligro).

text-secondary (Texto secundario).

text-dark (Este texto es gris oscuro).

text-body (Color de texto predeterminado (generalmente negro)).

text-light (Este texto es gris claro (en fondo blanco)).

text-white (Este texto es blanco (en fondo blanco)).
